= `hci secret echo`

`hci secret echo [OPTIONS]`

Assemble a secret and print it to stdout, ready for inclusion as a value in the
`secrets.json` dict.

== Parameters

[[option-project]]
=== `--project SITE/ORG/REPO`

Optional. Generate a condition expression that only allows access to the secret from the
default branch of specified repo.
